Practical Uses of Pointers in C

Pointers are a powerful feature in C that allow efficient memory management, data manipulation, and flexible program structures.Below are some practical use cases of pointers

1.Dynamic Memory Allocation

Pointers are used with malloc() and free() to allocate and release memory dynamically.

Example: Allocating memory for an array


    #include<stdio.h>
    #include<stdlib.h>
    
    int main(){
        int *arr;
        int n=5;

        //Allocating memory for an array of 5 integers
        arr=(int*)malloc(n*sizeof(int));

        //Storing values
        for(int i=0;i
<
n;i++){
            arr[i]=i+1;
        }

        //Printing values
        for(int i=0;i
<
n;i++){
            printf("%d",arr[i]);
        }

        //Freeing memory
        free(arr);

        return 0;
    }

2.Implementing Linked Lists

Linked lists rely on pointers to create dynamic data structures where each node contains a pointer to the next node.

Example: Singly Linked List


    #include<stdio.h>
    #include<stdlib.h>

    //Define a node
    struct Node{
        int data;
        struct Node* next;
    };

    //Function to create new node
    struct Node* createNode(int value){
        struct Node* newNode=(struct Node*)malloc(sizeof(struct Node));
        newNode->data=value;
        newNode->next=NULL;
        return newNode;
    }

    int main(){
        struct Node* head=createNode(10);
        head->next=createNode(20);
        head->next->next=createNode(30);

        //Print list
        struct Node* temp=head;
        while(temp!=NULL){
            printf("%d -> ", temp->data);
            temp=temp->next;
        }
        printf("NULL\n");

        return 0;
    }
                

    10 -> 20 -> 30 -> NULL

4.Pointer-Based String Manipulation

Pointers make string handling more efficient.

Example: String Copy Using Pointers


    #include<stdio.h>

    void copyString(char *source, char *destination){
        while(*source){
            *destination=*source;
            source++;
            destination++;
        }
        *destination="\0"; //Null-terminate the string
    }

    int main(){
        char source[]="Nyahello~";
        char destination[50];

        copyString(source,destination);
        printf("Copied String: %s\n",destination);

        return 0;
    }

5.Array Manipulation with Pointers

Pointers allow efficient traversal and modification of arrays.

Example: Using Pointers to Iterate Over an Array


    #include<stdio.h>

    int main(){
        int arr[]={10,20,30,40,50};
        int *ptr=arr;

        for(int i=0;i<5;i++){
            printf("Element %d: &d\n",i,*(ptr+i));
        }

        return 0;
    }

6.Passing Pointers to Functions(Call by Reference)

Pointers allow modifying function parameters directly.

Example: Swapping Two Numbers Using Pointers


    #include<stdio.h>

    void swap(int *a,int *b){
        int temp=*a;
        *a=*b;
        *b=temp;
    }

    int main(){
        int x=5,y=10;
        swap(&x,&y);
        printf("After Swap: x = %d, y = %d\n",x,y); //Output: x = 10, y = 5
        
        return 0;
    }
