What is pointer in C language?

1.Declaring a Pointer

A pointer in C is a variable that stores the memory address of another variable. Instead of holding a direct value, a pointer holds the location of a value in memory.
To declare a pointer, use the *(asterisk) symbol. 

    int *i;
    char *c;
    float *f;

2.Assigning an Address to a Pointer

Use the address-of operator(&) to assign the address of a variable to pointer. 

    #include<stdio.h>
    
    int main(){
        int num=10;
        int *ptr;

        ptr=#
        
        printf("Value of num: %d\n",num);
        printf("Address of num: %p\n",&num);
        printf("Pointer stores address: %p\n",ptr);
        printf("Value at pointer: %d\n",*ptr);

        return 0;
    }
                

    Value of num: 10
    Address of num: 0000007a545ff974
    Pointer stores address: 0000007a545ff974
    Value at pointer: 10

3.Dereferencing a Pointer

The *(dereferencing operator) is used to access the value stored at the address a pointer holds. 

    int a=20;
    int *p=&a;

    printf("%d",*p); //Output: 20

4.Pointer Arithmetic

You can perform arithmetic operations on pointers. 

    int arr[]=10,20,30;
    int *ptr=arr;

    printf("%d\n",*ptr); //10
    ptr++; //Move to next element
    printf("%d\n",*ptr); //20
    ptr++; //Move to next element
    printf("%d\n",*ptr); //30

5.Null Pointer

A pointer that does not point to any valid memory location is called a null pointer. 

    int *ptr=NULL;

6.Pointer to Pointer

A pointer to pointer stores the address of another pointer. 

    int a=5;
    int *p=&a;
    int **pp=&p; //Pointer to pointer

    printf("%d\n",**pp); //Output: 5

7.Dynamic Memory Allocation(malloc,free)

Pointers are essential in dynamic memory allocation. 

    int *ptr=(int*)malloc(sizeof(int)); //Allocates memory
    *ptr=42;
    printf("%d",*ptr); //Output: 42
    free(ptr); //Free allocated memory

8.Function Pointers

A pointer can also point to a function. 

    #include<stdio.h>
    
    void hello(){
        printf("Nyahello~\n");
    }

    int main(void){
        void (*funcPtr)(); //Declare function pointer
        funcPtr=hello; //Assign function address
        funcPtr(); //Call function using pointer

        return 0;
    }
